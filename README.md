# Braille_to_English_translator

This project is a web-based application that converts Braille images into English text using image processing and machine learning techniques.

## 🚀 Features

- Upload Braille image
- Detect Braille patterns
- Convert into readable English text
- Simple and user-friendly interface

## 🛠️ Technologies Used

- Python
- Flask
- OpenCV
- NumPy
- HTML/CSS
